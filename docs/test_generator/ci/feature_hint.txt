Area labels: <none>

### Bug Description

On the **Pipelines** list page, hovering over a pipeline row's **View** (eye) action shows a floating pipeline graph preview. This preview popup is positioned/anchored such that it **overflows past the right edge of the viewport** and gets clipped.

In the screenshot, the preview card containing the pipeline graph nodes (`logs - aws_e2e_logs` → `test[RAF]` → `logs - aws_e2e_logs`) extends beyond the right boundary of the browser window, so the right side of the preview (and parts of the node boxes) is cut off and not visible.

The preview should be fully contained within the viewport — it should flip/shift to the left of the cursor/row, or be constrained so it never renders off-screen.
