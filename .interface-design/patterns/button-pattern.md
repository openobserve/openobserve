# Button Pattern

## Primary Button
```vue
<q-btn
  data-test="primary-action-button"
  unelevated
  color="primary"
  :label="t('action.label')"
/>
```

## Secondary Button
```vue
<q-btn
  data-test="secondary-action-button"
  flat
  :label="t('action.label')"
/>
```

## Icon Button
```vue
<q-btn
  data-test="icon-action-button"
  icon="icon_name"
  flat
  dense
  round
/>
```
