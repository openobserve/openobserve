About this demo

A static multi-page site instrumented with OpenObserve RUM + Logs via the CDN.

What gets captured

  • A RUM view for every page you open (Home, Products, Checkout, About).
  • A continuous Session Replay across those navigations.
  • User actions, resource timing, long tasks, and errors.
  • Structured logs sent through the Logs SDK.

Generate a failed resource

This triggers a 404 resource error, visible in RUM error tracking.
